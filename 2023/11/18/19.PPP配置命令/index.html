18.PPP配置命令

文章发布时间:
2023-11-18

最后更新时间:
2023-11-24

认证方（ISP 侧）

  1. AAA 创建本地用户
  2. 进入运营商的接口开启 PAP 或者 CHAP 认证
  3. 可选配置（为接入的用户分配 IP 地址）
  • 创建地址池
  1. 接口调用地址池
  • 完成以上配置后，需要重启接口

CHAP 认证

  • 如果运营商的接口配置了用户名，则用户接口可以不用配置密码
  1. 运营商
  2. 用户设备只需要接口开启用户名，并使用全局密码即可
  • 如果运营商的接口没有用户名
  1. 运营商
  2. 用户必须在接口配置密码，使用接口密码来通过认证

用户侧配置（被认证方）

  1. 配置运营商提供的账号密码，通过身份认证
  2. 如果运营商侧配置了地址池，则用户可以获取公网 IP 地址 当配置完账户信息和地址协商后，需要重启接口 重新协商后，可以通过
  3. 内网与外网互访的配置
  4. 接口调用 NAT （Easy ip）
  5. 配置访问外网服务器的 IP 路由
